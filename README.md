#  Terminal Car Racing Game

A fast-paced car racing game that runs right in your terminal — built with pure Python!

##  How to Play
```bash
python car_game.py
```

| Key | Action     |
|-----|------------|
| W   | Move Up    |
| S   | Move Down  |
| A   | Move Left  |
| D   | Move Right |
| Q   | Restart    |
| E   | Exit       |

## Requirements

- Python 3.x
- Windows OS (uses `msvcrt` for keyboard input)
